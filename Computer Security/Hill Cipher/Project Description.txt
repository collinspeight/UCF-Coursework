Project Description: Write a program that encrypts the plain-text in a given file using the Hill cipher with 
a Hill matrix ranging from 2x2 to 9x9 in size.

Timeline: Approx. 2 weeks

NOTES:
	- Completed individually
	- The input is taken from the command line and consists of the name of the file storing the encryption 
	  key followed by the name of the file to be encrypted.
